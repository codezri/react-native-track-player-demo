# react-native-track-player-demo
A music track player app with a pre-defined playlist


## How to run the app?

- Download or clone the repository.
- Install dependencies:
```bash
npm install
# --- or ---
yarn install
```
- Run the app:
```bash
npx react-native run-android
# --- or ---
npx react-native run-ios
```
